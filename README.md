# 4IF-MACHINE-LEARNING
Machine Learning project for 4th year students at INSA Lyon

## Project description
The goal of this project is to get the best accuracy on the 2 datasets provided by Kaggle. 

## Datasets
The datasets are available on Kaggle:

- [INSA ML CLASSIFICATION](https://www.kaggle.com/competitions/insa-ml-2025-classification/data)
- [INSA ML REGRESSION](https://www.kaggle.com/competitions/insa-ml-2025-regression/data)
